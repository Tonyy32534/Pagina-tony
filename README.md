 Proyecto: Galería de Recetas Saludables

*Materia:* Desarrollo Web  
*Tecnologías:* HTML, CSS, JavaScript  
*Valor:* 20%  

1. Descripción general

Galería de Recetas Saludables es un sitio web interactivo diseñado para promover una alimentación consciente, nutritiva y deliciosa. Está dirigido a personas interesadas en cocinar recetas saludables, fáciles de preparar y organizadas por categorías, sin sacrificar el sabor.

 2. Objetivos

- Aplicar HTML semántico y diseño responsive.  
- Implementar animaciones y efectos visuales suaves.  
- Usar colores y tipografía adecuados al tema de salud y bienestar.  
- Crear una experiencia de usuario intuitiva y visualmente atractiva.  



 3. Estructura del proyecto

- /img – imágenes de las recetas usadas en el sitio.  
- index.html – página principal con todas las secciones integradas.  
- styles.css – hoja de estilos principal.  



 4. Paleta de colores

| Color | Código HEX | Significado |
|-------|------------|-------------|
| Verde Primario | #4CAF50 | Salud, naturaleza y frescura |
| Verde Secundario | #8BC34A | Armonía y crecimiento |
| Naranja Acento | #FF9800 | Energía y apetito |
| Blanco | #FFFFFF | Limpieza y pureza |
| Gris Texto | #333333 | Legibilidad y contraste |



 5. Tipografía

- *Títulos:* Poppins (moderna, elegante y legible).  
- *Texto:* Open Sans (alta legibilidad y claridad).  



 6. Animaciones y efectos

- Efecto fade-in con leve desplazamiento vertical en tarjetas de recetas al hacer scroll (Intersection Observer).  
- Transición suave en tarjetas y botones al pasar el cursor (hover).  
- Efecto de expansión en categorías de la sección de árbol.  



7. Características Interactivas

- Catálogo de recetas con funcionalidad de búsqueda en tiempo real.  
- Ordenamiento alfabético (A-Z o Z-A) de las recetas.  
- Árbol colapsable para navegar por categorías y subcategorías.  
- Botón de CTA en la sección de inicio para navegar rápidamente al catálogo.  



 8. Cómo ejecutar

1. Descargar la carpeta completa del proyecto.  
2. Asegurarse de que la carpeta /img con las imágenes esté en el mismo directorio que index.html.  
3. Abrir index.html en cualquier navegador moderno.  
4. Para visualizar las imágenes correctamente, es necesario tener los archivos de imagen en la carpeta /img.
